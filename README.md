## üîß How It Works

After you run the main script, follow on-screen prompts:

1. **Select Document Type**  
Choose whether you want a presentation (Beamer) or a paper (LaTeX).

2. **Input Your Data or R Scripts**  
You can add your data files or R code to the folder. The workflow will process them automatically.

3. **Review Suggestions**  
The system presents feedback generated by multiple review agents. You can accept, modify, or reject suggestions.

4. **Run Quality Checks**  
Step through automatic tests for formatting issues or data inconsistencies.

5. **Generate Final Output**  
Once everything checks out, the workflow builds your final PDF or presentation.

6. **Replication Report**  
Receive a summary report to help others verify your work.
